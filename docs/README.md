## 使用说明

### 变量说明
- `{{name}}`：文件名
- `{{type}}`：文件类型（文档、模板、图片等）
- `{{description}}`：文件描述
- `{{last_updated}}`：最后更新时间（格式：YYYY-MM-DD）
- `{{tags}}`：相关标签（使用 # 开头，多个标签用空格分隔）
- `{{importance}}`：重要性说明（高/中/低）
- `{{usage}}`：使用场景说明
- `{{related_links}}`：相关链接（Markdown格式）

### 使用方法
1. 复制此模板到目标目录
2. 根据实际文件情况，替换表格中的占位符
3. 删除不需要的行
4. 根据需要添加更多行

### 示例
```
| `basic_template.md` | 模板 | 基础文档模板 | 2025-02-28 | #模板 #基础 |
| `knowledge_template.md` | 模板 | 知识点文档模板 | 2025-02-28 | #模板 #知识 |
```

### 注意事项
- 文件名使用反引号包裹：`filename.md`
- 标签使用 # 开头：#模板 #文档
- 日期格式统一为：YYYY-MM-DD
- 描述信息要简洁明了
